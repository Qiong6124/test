# QtSQLite-ContactManager

Qt Widgets + SQLite 桌面通讯录管理应用

## 技术栈
- Qt 5.12.12 (Widgets + SQL)
- SQLite 3.x
- C++11

## 功能
- 联系人信息管理（姓名、电话、邮箱、地址、描述）
- 添加、编辑、删除联系人
- 按地址查询联系人
- 数据持久化存储

## 构建运行
```bash
# 环境要求
- Qt 5.12.x 及以上
- MinGW 32-bit 编译器

# 编译
qmake AddressBook.pro
make
```
